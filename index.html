Passo 1 di 3

Salva l'estrattore nei preferiti

Trascina il pulsante qui sotto all'interno della barra dei preferiti del tuo browser web.

🚀 GOG Scraper Engine
💡 Barra dei preferiti nascosta?
Mostrala subito premendo Ctrl + Shift + B (Su Mac: Cmd + Shift + B).
Passo 2 di 3

Esegui lo scraping e scarica il file

Apri il tuo account GOG e lancia il preferito appena salvato per generare l'archivio locale pulito.

2. Scarica il file: All'interno della scheda di GOG appena aperta, clicca sul preferito "GOG Scraper Engine". Lo script raccoglierà automaticamente i giochi e scaricherà direttamente un file sul tuo PC chiamato gog_rich_library.json.
Passo 3 di 3

Carica il file della libreria

Inserisci il file scaricato per generare all'istante la tua vetrina premium offline.

📁

Trascina qui il file gog_rich_library.json

oppure fai click per selezionarlo dalle cartelle

Rispetto assoluto della Privacy: L'elaborazione del file avviene localmente sul tuo browser. Nessun dato personale viene caricato o trasmesso in rete.
